# 和平美化包防封防冻结 【原版文件】


## 介绍

- 游戏文件（.Pak文件）：游戏的“**衣服**”
- 美化包（.Pak文件）：给游戏穿上了一套“奇装异服”（**让枪、衣服、车皮变好看**）。
- 原版（.Pak文件）：是游戏原本的**旧衣服**。



- **原版文件(pak)的作用**：
- 为了不被发现，这就像是在“奇装异服”外面又套了一件原来的衣服。游戏“保安”一看：**“哦，穿的是旧衣服，没问题”**，于是就放行了

### 使用教程

- 导入**“美化包或功能文件”**上游戏之后，游戏大厅有效果的话就退出登录，到登录界面，点击公告，打开MT管理器导入原版文件，然后再登账号进入游戏即可
